About My App

I chose to make an updated app of a program that we were given in class, Conway’s game of life. Conway’s game of life is a cellular automation, a program where you have a grid of squares (cells) that can either be "alive" or "dead". Conway’s game of life is a zero-player game, meaning as the game progresses in a series of generations, the screen displays cells, determined by a set of rules based on the of its neighbours (rules below). These rules determine whether a cell remains alive, dies, or becomes alive, creating complex patterns that can evolve over time.

New features and improvments 

  • Color Slider
    My first change was by adding a color picker slider. I chose to do this because I thought that it would be a cool way of making the user experience more fun and make it feel like you are able to customise the app/ game.The slider that was implemented was fairly simple. As it was located on the top of the screen under the start stops and had a slider with all available colours
  • Refresh Rate Slider
    I then came to the idea of adding the slider with the icons of a walking person and a running person this is because it is likely that people will be able to recognise what these symbols mean. I then had to decide where it was going to be placed on screen in the end I chose to add it above the color picker and below the start, stop, and clear buttons. This is because users would have a higher chance of seeing it as well as it not being in way of the game like in Conway
  • Added a link to the top of the app underneath the color slider. This link takes you to a website that talks/explains Conway's game of life. this link is made in a place that is accesable and keeps the modern theme of the app.
